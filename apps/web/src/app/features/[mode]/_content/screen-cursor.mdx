## How it works

Mode B captures your screen with an amber cursor pulse — no camera — into an MP4 · H.264 (AAC) recording.

### Step 1: Choose the mode

Open [/record](/record) and select **Screen + Cursor** from the mode picker.

### Step 2: Grant permissions

The browser will prompt for screen capture access. Camera is not required.

### Step 3: Record and download

Press record. The cursor pulse activates. When you stop, the file downloads as MP4 (H.264 + AAC), or WebM/VP9 as a fallback.

## Privacy

Fully client-side. No upload, no account.

## Codec output

`MP4 · H.264 (AAC)` — falls back to `WebM · VP9` if H.264 MediaRecorder is unavailable.

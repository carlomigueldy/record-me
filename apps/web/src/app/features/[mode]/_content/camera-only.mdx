## How it works

Mode C records your camera — no screen — into an MP4 · H.264 (AAC) file, downloaded directly to disk.

### Step 1: Choose the mode

Open [/record](/record) and select **Camera Only** from the mode picker.

### Step 2: Grant permissions

The browser will prompt for camera access. Screen capture is not required.

### Step 3: Record and download

Press record. When you stop, the file downloads as MP4 (H.264 + AAC), or WebM/VP9 as a fallback.

## Privacy

Fully client-side. No upload, no account. Nothing leaves your machine.

## Codec output

`MP4 · H.264 (AAC)` — falls back to `WebM · VP9` if H.264 MediaRecorder is unavailable.
